161 // Helper class to requested data.
162 // Most common data have dedicated getters, some need to be called with returned typa as a template.
163 // In general on either gets a gsl::span<const Type> via getter like e.g. getITSTracks()
164 // of a reference to particular track, i.e. getITSTrack(GlobalTrackID id).
165 // Note that random access like getITSTracks()[i] has an overhead, since for every call a span is created.
166 // Therefore, for the random access better to use direct getter, i.e. auto& tr = getITSTrack(gid)
167 // while for looping over the whole span first create a span then iterate over it.
